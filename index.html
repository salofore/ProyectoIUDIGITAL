Proyecto

Este módulo está destinado a la gestión de proyectos. Permite la creación, eliminación y actualización de proyectos existentes. Dentro de una organización o institución, un proyecto puede ser cualquier tarea o conjunto de tareas planificadas que se realizan para lograr un objetivo específico. En este caso implica la administración de detalles del proyecto, como su título, descripción, fechas de inicio y finalización, estado, entre otros..

Cliente

El módulo de Cliente está diseñado para manejar la gestión de clientes. Permite registrar, editar y eliminar información relacionada con los clientes que están involucrados en los proyectos. Los clientes pueden ser individuos, empresas u organizaciones. En este modulo se podría almacenar información como el nombre del cliente, detalles de contacto.

Etapa

Este módulo se centra en la gestión de las etapas de los proyectos. Las etapas representan diferentes fases o hitos dentro de un proyecto. Permitiría la creación, edición y eliminación de estas etapas para las etapas de vida del proyecto.

Tipo proyecto

Aquí se administra la clasificación de los proyectos según su tipo. Permite la creación, edición y eliminación de diferentes tipos de proyectos.

Universidad

El módulo de Universidad está destinado a la gestión de instituciones académicas o universidades. Permite la creación, edición y eliminación de registros relacionados con universidades asociadas o relevantes para los proyectos.
